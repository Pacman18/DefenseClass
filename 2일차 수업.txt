- 컨텐츠를만드는것은 한개만 만드는것이 아니라 여러개를 만드는 시스템에 적용한다고 생각한다. 

- 기본 디펜스 게임을 만들면서 프레임웍에 적용되게 하는 방법을 알아보자. 



a.  싱글톤으로 게임 매니저를 만들어보자 

싱글톤으로 만든 게임매니저에서 Scene를 변경하는 것을 적용하자. 


- 어때요?  이렇게 내가 작성하는것을보고 여러분이 배우는 방식인지 확인해봅시다  이렇게 배우는거 같아요? 


- 싱글톤에 대해서 모르면 설명하고가자. 


- 고블린 프리팹을 따로 만든다. 

- Agent 스크립트를 만든다.  ( 이것은 고블린에 붙인다 )

- 이 Agent를 관리할 AgentManager를만든다. 

- AgentManager는  Agent의 생성, 삭제를 관리한다. ( 게임의 기본인 A / B 팀의 Agent수에관한 이벤트를 보낸다 )
---> 로그로 확인해 봅시다. 



- GameStart 함수를  GameManager를 만들어서  호출이되면 Agent를 생성하는 코드를 만들어서 실행해봅시다. 

- Agent Team, State를 만들어서 하나씩 실행한다. 

- Agent가 없어지면 게임이 끝나는 로직을 만들어보자
